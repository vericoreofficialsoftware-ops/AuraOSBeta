# 💎 AuraOS: The Crystal Core
**AuraOS**, estetik ve performansın mükemmel dengesini arayan, modern 64-bit mimari üzerine inşa edilmiş minimalist bir işletim sistemidir. "Kristal Şeffaflığı" felsefesiyle tasarlanan arayüzü, en düşük donanımlarda bile akıcı bir deneyim sunmayı hedefler.

## 🌟 Öne Çıkan Özellikler

- **Modern 64-Bit Mimari:** Tamamen x86_64 tabanlı, geleceğe hazır çekirdek.
- **Kristal Kullanıcı Arayüzü (CUI):** Glassmorphism (cam efekti) odaklı tasarım dili.
- **Ultra Hafif:** 1 GB RAM ile tam performans, düşük kaynak tüketimi.
- **Şeffaf Geliştirme:** Her satırı temiz ve anlaşılır C ve Assembly kodu.

## 🛠 Teknik Mimari

| Bileşen | Detay |
| :--- | :--- |
| **Mimari** | x86_64 (64-bit) |
| **Hedef RAM** | 1 GB (Minimum 512 MB) |
| **Bootloader** | GRUB 2 (Multiboot2 Destekli) |
| **Arayüz Stili** | Glassmorphism / Blur Effects |
| **Kernel Tipi** | Monolithic / Micro-kernel Hybrid |

## 🚀 Geliştirme Süreci

AuraOS, şu anda aktif geliştirme aşamasındadır. Bulut tabanlı derleme (GitHub Actions) kullanılarak sürekli güncellenmektedir.

### Derleme Gereksinimleri
Yerel olarak derlemek isterseniz şunlara ihtiyacınız olacak:
- `gcc-x86-64-linux-gnu`
- `nasm`
- `grub-common`
- `xorriso`
